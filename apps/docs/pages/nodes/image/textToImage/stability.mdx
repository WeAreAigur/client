## Name
`stabilityTextToImage`

## Description

Returns an image generated from the given text prompt in the form of an `ArrayBuffer`.

## API Key

Uses the `stability` API key:

```ts index.ts
const aigur = createClient({
  apiKeys: {
    stability: process.env.STABILITY_API_KEY
  }
})
```

## Example

```ts index.ts
import { stabilityTextToImage } from '@aigur/client';

//...
flow.node(stabilityTextToImage, () => ({
  text_prompts: [
    {
      text: 'a green apple'
    }
  ]
})) // --> {result: ArrayBuffer}
```
