## Name
`stringToArrayBuffer`

## Description

Converts a base64 string to an ArrayBuffer

## Example

```ts index.ts
import { stringToArrayBuffer } from '@aigur/client';

//...
flow.node(stringToArrayBuffer, () => ({
  string: someBase64String,
})) // --> {arrayBuffer: ArrayBuffer}
```
